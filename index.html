¿Qué aprendimos?

Durante esta clase aprendimos a como desplegar nuestras paginas estaticas mediante la herramienta de Azure Static Web App con el uso de un repositorio de GitHub


¿Cómo se lleva el proceso para subir una página?

Con el uso de una cuenta de Azure, un repositorio de GitHub en donde tengamos nuestros archivos html, css o js, creamos un recurso de Static Web App en Azure y realizamos la implementacion, en cuestión de segundos el sitio web estará en la red.
